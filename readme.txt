A minimal, earthy WordPress theme for Thanchi Eco Resort - a wooden eco stay in the hills of Thanchi, Bandarban, Bangladesh.

== Description ==

Thanchi Eco Resort theme is designed specifically for the eco-friendly hotel located in Thanchi, Bandarban, Bangladesh. The theme features:

* SEO optimized with Schema.org markup
* WCAG accessibility compliant
* Minimal, earthy, breathable UI design
* WooCommerce compatible
* Hotel booking plugin compatible
* Blog ready
* Fast loading
* Responsive design

= Brand Colors =

* Forest Green: #1F3A2E
* Earth Brown: #9C6B3F
* Mist Background: #F4F3EF
* Text: #222222
* White: #FFFFFF

= Typography =

* Headings: Playfair Display
* Body: Inter

== Installation ==

1. Upload the theme folder to `/wp-content/themes/`
2. Activate the theme through the 'Themes' menu in WordPress
3. Create the following pages and assign templates:
   - Home (set as Front Page)
   - Rooms (use "Rooms Page" template)
   - Restaurant (use "Restaurant Page" template)
   - Shop (use "Shop Page" template)
   - About (use "About Page" template)
   - Contact (use "Contact Page" template)
   - Blog (set as Posts Page)

== Required Pages ==

Create these pages in WordPress:

1. **Home** - Set as static front page
2. **Rooms** - Template: Rooms Page
3. **Restaurant** - Template: Restaurant Page
4. **Shop** - Template: Shop Page
5. **About** - Template: About Page
6. **Contact** - Template: Contact Page
7. **Blog** - Set as posts page

== Recommended Plugins ==

* WooCommerce - For the shop functionality
* Hotel Booking plugin (e.g., MotoPress Hotel Booking, WP Hotel Booking)
* Contact Form 7 or WPForms - For the contact form
* Yoast SEO or Rank Math - For enhanced SEO
